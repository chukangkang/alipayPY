## 🔐 安全特性

### MD5 签名算法

与 New-API **完全兼容**的签名方式：

```
1. 过滤 sign、sign_type 和空值
2. 按 key 字母升序排序
3. 拼接：key1=value1&key2=value2&...&keyN=valueN
4. 末尾追加商户密钥（无 & 分隔符）
5. 计算 MD5 哈希值（小写十六进制）
```

### 多层验证

- ✓ 商户 ID 验证
- ✓ MD5 签名验证
- ✓ 支付宝 RSA 签名验证
- ✓ 订单幂等性检查
- ✓ 金额校验
